# DASH Humanoid Robot Project

This is the source code for the DASH Humanoid Robot Project website. It collects the materials needed to create the DASH Humanoid Robot and demonstrations of its capabilities.

The [Nerfies website](https://nerfies.github.io) was used as a template.
